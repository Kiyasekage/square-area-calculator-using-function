# Square Area Calculator

A simple Python program that calculates the area of a square using a user-defined function. The program prompts the user to enter the length of a square's side, passes the value to a function that computes the area by squaring the side length, and then displays the calculated area. This project is ideal for beginners who want to practice creating and calling functions in Python.

## Features

* Accepts user input for the length of a square's side.
* Uses a custom function to calculate the area.
* Computes the area by squaring the side length.
* Displays the calculated area in a clear and readable format.
* Demonstrates the use of functions, user input, and mathematical operations in Python.

## How to Run

1. Make sure Python 3 is installed.
2. Run the script:

   ```bash
   python square_area.py
   ```
3. Enter the length of the square's side when prompted.
